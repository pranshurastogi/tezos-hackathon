Tezos Hackathon : SmartPy
===


Prerequisites
---

Python 3.6+

Quick Start
---
 
 - ``npm run smartpy:install`` Install SmartPy basic
 - ``npm run smartpy:test -- example.contract`` Run the tests for a specific smart contract
